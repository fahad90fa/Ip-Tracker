# Contributing to VIPER-EYE

Thank you for your interest in improving VIPER-EYE.

## Code of Conduct
- Be respectful, professional, and collaborative.
- Avoid disruptive behavior, harassment, or destructive changes.
- Keep discussions focused on improving the project.

## How to Contribute
1. Fork the repository.
2. Create a feature branch.
3. Make your changes with clear commit messages.
4. Run tests before submitting a pull request.
5. Open a pull request with a concise summary and validation notes.

## Pull Request Guidelines
- Keep changes focused and well-documented.
- Add tests for new logic when possible.
- Ensure the project still compiles and passes tests.
- Explain the problem being solved and the expected outcome.
